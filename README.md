# Netflix Task Submission

This project is a **Netflix Sign Up Page** created using **HTML5** and **external CSS**.

The HTML file (`index.html`) links to an external stylesheet located at `./assets/css/main.css`.  
All styling is handled through this external CSS file.

## Key Points

- Uses **HTML and external CSS only**
- No internal `<style>` tag
- No inline CSS for styling
- Includes a sign-up form with inputs, checkbox, and button
- Netflix logo created using SVG

## Purpose

This project is created as a **task submission** to demonstrate:

- Proper HTML structure
- Use of external CSS
- Clean and simple UI design
